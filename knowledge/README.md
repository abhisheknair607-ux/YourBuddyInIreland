# Knowledge Folder

Drop your local source documents here.

The chatbot checks these files first before using web search.

## Supported file types

- `.pdf`
- `.docx`
- `.md`
- `.txt`

## Suggested use

- Visa guides
- University brochures
- Accommodation notes
- Loan documents
- Research summaries
- Course comparison notes

## How it works

1. The app searches this folder first.
2. If it finds relevant excerpts, Gemini answers from those excerpts only.
3. If the local files do not cover the question, Gemini falls back to Google Search.

## Tip

Use clear file names like:

- `ireland-visa-guide.pdf`
- `student-housing-dublin.docx`
- `loan-options-india-to-ireland.md`
- `msc-data-analytics-notes.txt`
